## Overview
Todolist Management System is a PHP and MySQL-based application designed to help users manage their daily tasks efficiently. This system allows users to create, update, and delete tasks, providing a simple yet effective way to organize their to-do lists.

## Features
- User authentication and session management
- Create, read, update, and delete (CRUD) tasks
- Task categorization and prioritization
- Responsive design with CSS
- Database integration with MySQL

## Files and Directories
- **systemtodo/**: Main directory containing all the project files.
- **.gitattributes**: Git configuration file.
- **desktop.ini**: Windows desktop initialization file.

## Installation and Setup
1. Clone the repository:
    ```sh
    git clone https://github.com/elif1906/Todolist-management-system.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Todolist-management-system/systemtodo
    ```
3. Set up the database:
    - Create a MySQL database named `todolist`.
    - Import the `todolist.sql` file located in the project directory into your database.
4. Configure the database connection:
    - Update the `db.php` file with your database credentials.

5. Run the application:
    - Start your local server (e.g., XAMPP, WAMP).
    - Navigate to `http://localhost/Todolist-management-system/systemtodo` in your web browser.

## Usage
- **Sign Up/Login**: Create a new account or log in with existing credentials.
- **Manage Tasks**: Add, edit, or delete tasks from your to-do list.
- **Categorize Tasks**: Organize tasks by categories for better management.
- **Prioritize Tasks**: Set priority levels for tasks to focus on important items.

## Troubleshooting
- **Database Connection Issues**: Ensure that your database credentials in `db.php` are correct.
- **Server Issues**: Make sure your local server is running and properly configured.
